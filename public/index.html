How to Play

  1. Create or Join - One person creates a room and shares the 4-letter code. Others join with that code.
  2. See a Word - Everyone sees the same prompt word (like "hot").
  3. Think Fast - Type the first word that comes to mind (like "dog").
  4. Match to Score - Players who submit the same answer earn points!
  5. Speed Bonus - First to submit a matching answer gets +50 bonus points.

Tip: Think of common word pairs! "Hot" → "dog", "Ice" → "cream"
